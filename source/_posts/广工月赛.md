## J
- 分析
这题目也不错，但其实看代码找规律就行了

[题目链接](https://ac.nowcoder.com/acm/contest/119605/J)

![代码](/images/J.png)

代码:
```yaml
#include<bits/stdc++.h>
using namespace std;
#define ll long long
int main(){
    ll n; cin>>n;
    ll s=sqrt(n);
    if(s*s!=n) s++;
    ll last=n%s;
    for(int i=last;i>=1;i--) cout<<i<<" ";
    for(int j=1;j<=n/s;j++){
        for(int i=last+j*s;i>last+(j-1)*s;i--) cout<<i<<" ";
    }
}
```
